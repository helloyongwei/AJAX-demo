# 安装
使用git clone命令安装:
```git
git clone git@github.com:helloyongwei/AJAX-demo.git
```

# 运行
进入AJAX-demo目录, 并执行:
```node
node server.js 8888
```
8888是端口, 你可以自行指定端口

在浏览其输入如下URL即可:
```
http://localhost:8888
```
你会看到一个页面:
![响应前的图片](/imgs/ajax.png)

打开控制台, 点击页面的按钮. 你会看到控制台(图中红框部分)输出如下内容:
